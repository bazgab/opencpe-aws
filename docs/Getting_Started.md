## Getting Started

OpenCPE is an open source Cloud-Native Policy Engine focused on notifying resource owners on non-compliant resources.

With the goal of simplicity and ease of maintenance, its' design is based on a single configuration file, and all the environment is **Cloud Specific**, meaning that each Public Cloud Provider has a different associated version of OpenCPE with it, allowing all of the resource gathering, manipulating and mailing to be constrained within the same CSP.

### OpenCPE AWS

This tutorial will show how to simply setup OpenCPE with AWS.

### Requirements

- `opencpe-aws` installed
- AWS Account with [IAM Identity Center](https://aws.amazon.com/iam/identity-center/) setup and configured
